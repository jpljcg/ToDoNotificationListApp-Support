# ToDo Notification List – Support

Need help with the app?  
Contact: **todonotificationlist.support@myappcatalogue.com**

## What this app does
ToDo Notification List helps you create recurring reminders and keep routines on track.

## Features
- Daily, weekly, and monthly recurring schedules
- Custom intervals (every 15 minutes up to 12 hours)
- Cloud sync across devices (subscription)

## FAQ

### 1) I don’t see subscription prices
- Make sure you are online.
- Close and reopen the app.

### 2) Restore Purchases asks for Apple login
This can happen on iOS and is expected behavior for restore/purchase flows.

### 3) Notifications are not firing
- Check notification permissions.
- Disable Focus/Do Not Disturb if needed.
- Ensure app notifications are enabled in Settings.
